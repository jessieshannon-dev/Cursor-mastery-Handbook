# Complete Cursor Mastery Handbook Website

This is a static website you can share with Brandon or upload to Vercel.

## Files

- `index.html`
- `styles.css`
- `script.js`

## Open Locally

Double-click `index.html`.

## Deploy to Vercel

1. Create a GitHub repository.
2. Upload these files.
3. Go to Vercel.
4. Import the repository.
5. Click Deploy.
6. Share the Vercel link.

## Edit

Open the folder in Cursor and tell Cursor what you want changed.
